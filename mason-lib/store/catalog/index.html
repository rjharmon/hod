Description

The store catalog manager provides an easy-to-use interface for viewing, organizing, updating and maintaining your online store catalog. It provides multiple ways to filter your product line and multiple ways to view the product line, as well as viewing the photographs associated with your products. It even lets you upload your own photos!

Usage

To use the store catalog manager, first ensure that the administrators have enabled the database to use your catalog. Next, define a /store/catalog/login.m that authenticates the incoming user to protect access to the catalog. You may wish to use /password.mason for this purpose.

<% <
return 0 unless \$m->comp('/password.mason',
   realm => 'Store Manager',
   allow => { qw(
      admin ei436dmns8
   ) },
   production => 1
);
return 1;
